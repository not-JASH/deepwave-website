# Source notes and translation log

This website draft was based on the uploaded recommendations and the expanded PDF report. It also references official public pages for contemporary corporate minimalism examples. These notes are included so the design logic remains traceable.

## Uploaded sources translated into the site

### Corporate Website Recommendations for a Research Company

Key recommendations implemented:

- A research company website should feel credible, rigorous, clear, and quietly distinctive.
- The site should be a research operating system, not a brochure.
- The research library should be central, with search and filters by topic, method, format, and date.
- Report cards should include title, finding, tags, date, author/team, format, and CTA.
- The homepage should move through hero, domains, featured insight, proof, library, methods, outcomes, and contact.
- Visuals should be data-informed: grids, cells, chart linework, annotations, and statistical motifs.
- Accessibility should be visible through WCAG-minded contrast, 18–20px body type, focus states, keyboard navigation, chart summaries, and HTML report pages.

### Expanded Contemporary Corporate Minimalism Report

Key principles implemented:

- Minimalism as governance: the CSS uses tokens for color, type, spacing, radius, motion, and surfaces.
- Restrained distinctiveness: the custom wave-grid motif is the single memorable rule.
- Type as infrastructure: the system defines sans, serif, and mono roles rather than relying on one face.
- Motion as behavior: the library filtering, scan motif, and chart reveal imply discovery and evidence-building rather than spectacle.
- Warmth compensation: the draft uses warm neutral backgrounds, researcher profiles, human copy, and case-study framing.
- Accessibility evidence: low contrast is avoided; secondary text is dark enough to remain readable.

## Official pages reviewed

- Lippincott / Nokia — segmented wordmark and identity system as B2B repositioning.
- Pentagram / PayPal — bespoke typeface, streamlined palette, and payment-behavior motion.
- Pentagram / Reddit and Reddit Inc. — Reddit Sans, conversation-bubble motif, accessible type rationale, and platform-specific identity.
- Pentagram / Navan — one-shape curve system that becomes narrative and motion infrastructure.
- Koto / Bolt — product benefit expressed as visual speed and one-click motion.
- Pentagram / Igluu — warm minimalism, serif-led trust, embedded search icon, and people-centered imagery.
- WebAIM Million 2026 — low-contrast text as a common web accessibility failure.
- W3C WCAG 2.2 — 4.5:1 normal text contrast, 3:1 large text contrast, and resize-text requirements.
- IBM Design / Plex — corporate type as open-source global infrastructure.
